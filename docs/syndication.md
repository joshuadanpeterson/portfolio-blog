# Syndication Sources

The portfolio syndicates external writing through first-party source feeds where possible.

- Medium uses `https://medium.com/feed/@joshpeterson`.
- Attention Wars uses `https://attentionwars.substack.com/feed`.
- Steemit uses the public JSON-RPC API at `https://api.steemit.com`.

The old `rss.app` wrapper for Steemit has been removed from runtime code.

X is intentionally out of scope for this pass. If X syndication is added later, only X Articles should be considered for the portfolio; do not add broad `x.com/jdpeterson` post syndication as a substitute. Future X Articles work should choose either an official X API/link-out path or an approved article extraction provider with an explicit cost and credential gate.
